# naginata-keymap-config
## 薙刀式 keymapper 設定

## 新下駄配列 設定
### Windows msiでインストール
https://github.com/houmain/keymapper?tab=readme-ov-file#windows

トライアイコン メニューからConfigurationを選択
開いたketmapper.confに
sin_geta_Windows.confをテキストをコピーして保存します
トライアイコンからReloadで、読み込み


### ポータブル zip
https://github.com/houmain/keymapper/releases
keymapper-5.3.1-Windows-x86_64.zip
を展開して

その展開したフォルダにkeymapper.confを新規作成して
sin_geta_Windows.confをテキストをコピーして保存します

keymapperd.exeとkeymapper.exeを
実行します

## 新下駄配列の入力
ESC ＋ スペースキーで入力開始
無変換で英数入力に戻ります
